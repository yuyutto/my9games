# 私を構成する9つのゲーム

自分を形作った9つのゲームを選んで、画像として出力・保存できるローカルWebアプリ。

![スクリーンショットのイメージ: 3×3グリッドにゲームカバーが並ぶ]

## 機能

- **3×3グリッド**でゲームカバーを並べて表示
- ゲームの検索・選択（Steam / IGDB の2ソース対応）
- フリーゲームや自炊画像にも対応する**手動入力モード**
- 各ゲームへの**ひとことコメント**入力
- コメントの**ネタバレ隠し**（ぼかし表示）
- グリッドを**PNG画像として保存**
- 選択内容はブラウザの localStorage に**自動保存**

## 必要なもの

- Python 3.x（標準ライブラリのみ使用、追加インストール不要）
- インターネット接続（ゲーム検索・カバー画像取得に使用）

## セットアップ

### 1. リポジトリをクローン

```bash
git clone https://github.com/yourname/my9games.git
cd my9games
```

### 2. IGDB を使う場合のみ：APIキーを設定

Steam 検索だけ使う場合はこの手順は不要です。

1. [Twitch Developer Console](https://dev.twitch.tv/console) にアクセス（Twitchアカウントでログイン）
2. 「アプリケーションを登録」→ 任意の名前、OAuth Redirect URL は `http://localhost`、カテゴリは `Application Integration`
3. 取得した **Client ID** と **Client Secret** を `my9games.py` の先頭に記入：

```python
IGDB_CLIENT_ID = "ここに記入"
IGDB_CLIENT_SECRET = "ここに記入"
```

### 3. 起動

```bash
python my9games.py
```

ブラウザが自動で開きます。開かない場合は http://localhost:8080 にアクセスしてください。

終了するには `Ctrl+C`。

## 使い方

### ゲームを選ぶ

グリッドの「＋ ゲームを選ぶ」をクリックすると検索モーダルが開きます。

| タブ | 説明 |
|------|------|
| **Steam** | Steam に登録されているゲームを検索（キー不要） |
| **IGDB** | 35万以上のゲームDB（コンソール・レトロ・モバイルも対応） |
| **手動入力** | タイトルを直接入力、カバー画像ファイルを選択して登録 |

> **IGDB で見つからないゲーム**（フリーゲーム・同人ゲームなど）は「手動入力」タブを使ってください。

### コメントを書く

ゲームを選択するとタイトル下にテキストエリアが表示されます。自由にコメントを入力できます。

「ネタバレ隠す」ボタンを押すとコメントがぼかし表示になり、画像出力にも反映されます。

### 画像を保存

下部の「📷 画像を保存」ボタンで `my9games.png` がダウンロードされます。

- ボタン類（「ネタバレ隠す」「✕」など）は画像に含まれません
- ネタバレ隠し中のコメントはぼかしたまま出力されます
- 画像は2倍解像度（Retina対応）で出力されます

## ファイル構成

```
my9games.py        # サーバー本体（HTML/CSS/JSをすべて内包）
my9games.html      # 旧バージョン（Steam非対応のスタンドアロンHTML版）
```

## APIの仕組み

`my9games.py` は Python の標準ライブラリだけで動く軽量HTTPサーバーです。
Steam・IGDB への API リクエストはすべてサーバー側で行うため、ブラウザの CORS 制限を回避しています。

| エンドポイント | 内容 |
|---|---|
| `GET /` | HTMLページ配信 |
| `GET /api/search?q=` | Steam Store 検索 |
| `GET /api/cover?id=` | Steam カバー画像取得（縦長 600×900） |
| `GET /api/igdb_search?q=` | IGDB 検索 |
| `GET /api/igdb_cover?url=` | IGDB カバー画像取得 |
